Detailed Description

Class representing an OSC Time Tag.

TimeTag class for handling OSC time tags. OSC time tags are represented in NTP format with seconds since 1900 and fractional seconds.

OSC Time Tags are 64-bit fixed-point numbers representing time in NTP format (seconds since Jan 1, 1900).
